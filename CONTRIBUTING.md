# Contributing

Thanks for your interest in contributing!

## Adding a New Language

1. Create a new JSON file in `affirmations/` (e.g., `fr.json` for French)
2. Add 20+ affirmations as a JSON array of strings
3. Add the language code to the `affirmations` object in `index.js`
4. Submit a pull request

## Adding More Affirmations

1. Edit the appropriate language file in `affirmations/`
2. Add your affirmations to the array
3. Submit a pull request

## Running Locally

```bash
npm install
npm start
```

## Guidelines

- Keep affirmations fun and positive
- No offensive or inappropriate content
- Test your changes before submitting
